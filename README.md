# image2dxf-hough

1. Put images (PNG/JPG/TIFF) into the `input/` folder.
2. Install dependencies:
   python -m pip install -r requirements.txt
3. Run:
   python src/convert_to_dxf.py
   (or specify folders: python src/convert_to_dxf.py path/to/input path/to/output)
4. DXF files appear in the `output/` folder.
5. Open DXF in AutoCAD / LibreCAD and clean up as needed.
